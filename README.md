# 🛒 NexKart — Ecommerce Application

🔗 **Live Demo:**
https://nexkart-ecommerce-app.vercel.app

---

##  Overview

**NexKart** is a modern ecommerce web application that allows users to explore products, search items, and manage their shopping cart with a smooth and responsive user experience.

---

## ✨ Features

*  Product Listing (Dynamic UI)
*  Search Functionality (Real-time Filtering)
*  Add to Cart System
*  Cart Management (Add / Remove Items)
*  Fully Responsive Design (Mobile + Desktop)
*  Fast Performance (Optimized with Vite)
*  Client-side Routing using React Router

---

## 🛠️ Tech Stack

* **Frontend:** React.js, Vite
* **Styling:** Tailwind CSS / CSS
* **Routing:** React Router
* **Database:** Firebase Firestore
